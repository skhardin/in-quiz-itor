# in-quiz-itor

Python CLI that takes a .csv file containing vocabulary ```key, definition``` as input, quizzes user and returns a score. 

## Main Menu

```
==============================
Quiz Menu:
==============================
1. Start Quiz
2. Modify Quiz Content
3. Exit
==============================
Please choose an option (1-3): 
==============================
```

## Quiz Format

### Level 1: Pre-assessment

Description: Shuffles content and asks user to respond with correct ```key``` given a ```defintion```. Resulting score decides next round. 


### Level 2: Fill-in-the-blank (Easy)
